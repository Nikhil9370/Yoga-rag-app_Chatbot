## Overview
This is a full-stack RAG-based wellness application that answers yoga and fitness queries.

## Tech Stack
- Frontend: React
- Backend: Node.js / Express
- Database: MongoDB
- RAG: Embeddings + Vector Search

## RAG Flow
1. User asks question
2. Query converted into embeddings
3. Relevant yoga documents retrieved
4. LLM generates final answer using retrieved context

## Safety Handling
- Medical disclaimer shown
- Unsafe queries blocked
- Source transparency

## Setup Instructions
1. Clone repo
2. Install dependencies
3. Add .env
4. Run backend
5. Run frontend
